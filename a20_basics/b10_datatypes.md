# Available types

1. `int`
2. `uint`
3. `float64`
4. `string`
5. `bool`
6. `err`
7. `struct`


### `string`

- Double quotes or backticks are allowed
```
var str = "some value"
var str = `mutli line 
string`
```

### struct

Custom structure can be created using `type` & `struct` keyword

```go
type User struct {}
```
